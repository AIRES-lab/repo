# AIRES Lab Website

This repository contains the public website for AIRES Lab: Artificial Intelligence
Research for Experimental Security.

## Local Preview

Open `index.html` in a browser. The site is static HTML, CSS, and JavaScript, so
it does not require a build step.

## GitHub Pages

Publish from:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`

The project site URL should be:

```text
https://aires-lab.github.io/repo/
```
